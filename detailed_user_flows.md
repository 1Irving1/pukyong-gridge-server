### 상호작용 플로우
```
사용자 액션                    시스템 응답                    백엔드 처리
─────────────────────────────────────────────────────────────────────────

1. 전화번호 입력창 타이핑
   ├─ 20자 이하 입력          → 입력값 표시 + ✓ 아이콘        → 형식 검증
   ├─ 20자 초과 입력          → 20자까지만 입력, 나머지 무시    → 길이 제한
   ├─ 엔터 입력               → 줄바꿈 없음                   → 이벤트 무시
   ├─ 형식 불일치             → X 아이콘 표시                → 정규식 검증
   └─ 중복 전화번호            → X 아이콘 표시                → DB 중복 확인

2. 이름 입력창 타이핑
   ├─ 20자 이하 입력          → 입력값 표시 + ✓ 아이콘        → 길이 검증
   ├─ 20자 초과 입력          → 20자까지만 입력               → 길이 제한
   └─ 엔터 입력               → 줄바꿈 없음                   → 이벤트 무시

3. 사용자명 입력창 타이핑
   ├─ 20자 이하 입력          → 입력값 표시 + ✓ 아이콘        → 길이 검증
   ├─ 20자 초과 입력          → 20자까지만 입력               → 길이 제한
   ├─ 엔터 입력               → 줄바꿈 없음                   → 이벤트 무시
   ├─ 중복 사용자명            → X 아이콘 표시                → DB 중복 확인
   ├─ 형식 불일치             → X 아이콘 표시                → 정규식 검증
   └─ 유효한 형식             → ✓ 아이콘 표시                → 검증 통과

4. 비밀번호 입력창 타이핑
   ├─ 20자 이하 입력          → ****** 표시 + ✓ 아이콘       → 길이 검증
   ├─ 20자 초과 입력          → 20자까지만 입력               → 길이 제한
   ├─ 엔터 입력               → 가입 버튼 클릭과 동일         → 이벤트 처리
   └─ 6자 이상 입력           → 가입 버튼 활성화              → 조건 검증

5. 비밀번호 표시/숨기기
   ├─ "비밀번호 표시" 클릭     → 실제 비밀번호 표시            → UI 상태 변경
   └─ "숨기기" 클릭           → ****** 표시                  → UI 상태 변경

6. 가입 버튼 클릭
   ├─ 모든 필드 유효          → 생일 선택 화면으로 이동        → 회원가입 진행
   ├─ 전화번호 형식 오류       → "휴대폰 번호가 정확하지 않습니다" → 오류 메시지
   ├─ 사용자명 형식 오류       → "사용자 이름에는 문자, 숫자..." → 오류 메시지
   └─ 사용자명 중복           → "사용할 수 없는 사용자 이름입니다" → 오류 메시지

7. 카카오 로그인 버튼 클릭
   ├─ 카카오 인증 성공        → 메인 화면으로 이동            → OAuth 처리
   ├─ 카카오 인증 실패        → "Login failed" 팝업          → 오류 처리
   └─ 신규 사용자             → 회원가입 화면으로 이동        → 추가정보 입력
```

```
   ├─ 드롭다운 열림            → 1일~31일 목록 표시            → 정적 데이터
   └─ 일 선택                  → 선택된 일 표시                → UI 상태 업데이트

3. 년도 선택 드롭다운 클릭
   ├─ 드롭다운 열림            → 1919년~2021년 목록 표시       → 정적 데이터
   ├─ 1919-2015년 선택         → 가입 버튼 활성화              → 연령 검증
   └─ 2016-2021년 선택         → 가입 버튼 비활성화            → 연령 제한

4. 가입 버튼 클릭
   ├─ 유효한 생년월일          → 이용약관 동의 화면으로 이동    → 회원가입 진행
   └─ 비활성화 상태            → 아무 동작 없음               → 버튼 상태 확인

5. 돌아가기 버튼 클릭
   └─ 클릭                    → 기본정보 입력 화면으로 이동    → 이전 단계 복귀

6. 로그인 텍스트 클릭
   └─ 클릭                    → 로그인 화면으로 이동          → 화면 전환
```

### 상호작용 플로우
```
사용자 액션                    시스템 응답                    백엔드 처리
─────────────────────────────────────────────────────────────────────────

1. "이용약관 3개에 모두 동의" 체크박스 클릭
   ├─ 비어있는 상태에서 클릭    → 3개 개별 약관 모두 체크        → 일괄 동의 처리
   ├─ 모두 체크된 상태에서 클릭 → 3개 개별 약관 모두 해제        → 일괄 해제 처리
   └─ 일부 체크된 상태에서 클릭 → 3개 개별 약관 모두 체크        → 일괄 동의 처리

2. 개별 약관 체크박스 클릭
   ├─ 개별 약관 체크           → 해당 약관 체크 표시            → 개별 동의 처리
   ├─ 개별 약관 해제           → 해당 약관 체크 해제            → 개별 해제 처리
   ├─ 3개 모두 체크           → "이용약관 3개에 모두 동의" 체크 → 마스터 체크 활성화
   └─ 3개 모두 체크           → "다음" 버튼 활성화              → 조건 충족 확인

3. "다음" 버튼 클릭
   ├─ 모든 필수 약관 동의      → 로그인 화면으로 이동            → 회원가입 완료
   └─ 필수 약관 미동의         → 버튼 비활성화 상태 유지         → 조건 미충족

4. "돌아가기" 버튼 클릭
   └─ 클릭                    → 생일 선택 화면으로 이동        → 이전 단계 복귀

5. "더 알아보기" 링크 클릭
   └─ 클릭                    → 해당 약관 상세 내용 표시        → 정적 콘텐츠
```

### 상호작용 플로우
```
사용자 액션                    시스템 응답                    백엔드 처리
─────────────────────────────────────────────────────────────────────────

1. 사용자명 입력
   ├─ 입력                    → 사용자명 표시                  → 입력 처리
   └─ 엔터 입력               → 다음 버튼 클릭과 동일           → 이벤트 처리

2. 전화번호 입력
   ├─ 입력                    → 전화번호 표시                  → 입력 처리
   └─ 엔터 입력               → 다음 버튼 클릭과 동일           → 이벤트 처리

3. "다음" 버튼 클릭
   ├─ 유효한 정보             → 비밀번호 재설정 화면으로 이동    → 인증 진행
   └─ 잘못된 정보             → 오류 메시지 표시                → 검증 실패

4. 새 비밀번호 입력
   ├─ 첫 번째 필드 입력        → 비밀번호 표시                  → 입력 처리
   ├─ 두 번째 필드 입력        → 비밀번호 표시                  → 입력 처리
   └─ 일치하지 않는 비밀번호   → 오류 메시지 표시                → 일치 검증

5. "비밀번호 재설정" 버튼 클릭
   ├─ 유효한 비밀번호          → 로그인 화면으로 이동            → 비밀번호 업데이트
   └─ 잘못된 비밀번호          → 오류 메시지 표시                → 검증 실패

6. "로그인으로 돌아가기" 클릭
   └─ 클릭                    → 로그인 화면으로 이동            → 화면 전환
```
